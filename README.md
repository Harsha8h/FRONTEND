# 🚗 Bootstrap Image Carousel Gallery

A responsive image gallery built using **HTML5**, **Bootstrap 4**, and **CSS**. The project showcases multiple image cards inside a Bootstrap carousel, allowing users to navigate through different slides using previous and next controls.

This project was developed to practice Bootstrap components, responsive layouts, and card-based UI design.

---

## 📌 Features

- Responsive Bootstrap Carousel
- Card-based image gallery
- Previous and Next navigation controls
- Responsive grid layout
- Bootstrap Cards
- Mobile-friendly design
- Simple and lightweight implementation

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 4.6
- JavaScript
- jQuery


Simply open **index.html** in your preferred web browser.
